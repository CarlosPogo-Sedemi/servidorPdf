¿Cómo funciona el procesador de CAD (.dwg / .dxf) en Python?
Manejabilidad de CAD en Local

1. El formato DXF (.dxf) (Abierto): Es sumamente manejable en Python. Usamos la librería ezdxf para leer directamente la geometría del plano (líneas, círculos, cotas) y matplotlib para renderizar el dibujo vectorial de forma nativa a una imagen y pegarla en el PDF.

2. El formato DWG (.dwg) (Propietario): Al ser binario y cerrado, no existe librería de código abierto nativa para Python que lo renderice sin licencias comerciales. El estándar recomendado es:

  • Opción Gratis: Usar la herramienta CLI gratuita ODA File Converter para automatizar por backend la conversión de DWG a DXF y luego leerlo con ezdxf.
  • Opción Directa: Usar pip install aspose-cad (que incluye soporte directo de lectura para DWG en Python mediante API).
Código fuente de extracción implementado:
import ezdxf
from ezdxf.addons.drawing.matplotlib import qsave

# 1. Cargar archivo DXF
doc = ezdxf.readfile("plano.dxf")
msp = doc.modelspace()

# 2. Renderizar plano completo a PNG
qsave(msp, "captura.png")

# 3. Insertar la captura en ReportLab
canvas.drawImage("captura.png", x, y, width, height)
